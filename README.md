DDoS-Firewall – игра о защите от кибератак

DDoS-Firewall — браузерная стратегическая игра, где игрок выступает в роли агента киберзащиты. В твоих руках — карты защитных действий, с помощью которых нужно отражать атаки хакерского сервера.

Легко расширяемая механика карт и атак для дальнейших апдейтов.
